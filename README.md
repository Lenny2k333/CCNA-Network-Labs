# CCNA Networking Lab Portfolio

This repository contains hands-on networking labs completed while studying for the Cisco CCNA.

Each lab focuses on real-world network configuration, troubleshooting, and design using Cisco Packet Tracer.

## Labs

### 1. Enterprise VLAN Network Segmentation
Technologies:
- VLANs
- Router-on-a-stick
- VLSM subnetting
- Inter-VLAN routing

### 2. VLAN Troubleshooting Lab
Diagnosing and repairing broken VLAN connectivity.

### 3. DHCP Network Configuration
Centralized address assignment using DHCP.

### 4. OSPF Routing Lab
Dynamic routing between multiple routers.

### 5. NAT Internet Access Lab
Allowing internal networks to access the internet using NAT.

### 6. Network Troubleshooting Scenario
Real-world troubleshooting across multiple networking layers.

Tools used:
- Cisco Packet Tracer
